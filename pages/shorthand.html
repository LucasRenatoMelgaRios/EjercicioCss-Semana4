Shorthand: Que es y para que sirve.

En CSS, el término "shorthand" se refiere a la capacidad de especificar múltiples propiedades relacionadas con un solo valor. Esto permite escribir código más conciso y fácil de leer. Los shorthand son atajos que combinan varias propiedades en una sola declaración. Algunos ejemplos comunes de shorthand incluyen:

Font Shorthand: font: Permite establecer varias propiedades de fuente en una sola declaración, como font-style, font-weight, font-size, line-height y font-family.

Ejemplo:

p {

font: italic bold 16px/1.5 "Times New Roman", serif;

}

Margin y Padding Shorthand:

margin: Combina margin-top, margin-right, margin-bottom y margin-left en una sola declaración.

padding: Combina padding-top, padding-right, padding-bottom y padding-left en una sola declaración

Ejemplo:

p {

margin: 10px 20px 10px 20px; /* arriba derecha abajo izquierda */

}

div {

padding: 5px 10px; /* arriba y abajo, derecha e izquierda */

}
