Selectores simples

Selector de etiqueta:

Estos selectores son los más simples y tienen especificidad 1, al utilzarlos aplicarán estilos a todas las etiquetas del mismo tipo

Ejemplo:

h3{

color: red

}

Este selector aplicará estidos a todas las etiquetas <h3>

Selector de clase:

Estos selectores tienen especificidad 10, se llaman mediante: ."Nombre de la clase" y son más usados con los estilos css

Ejemplo:

.encabezadoPrincipal{

color: black

}

Este selector aplicará estidos a todas las etiquetas con la clase "encabezadoPrincipal".

Selector de ID:

Estos selectores se llaman mediante #"nombreDeLaClase" aunque no es recomendado usarlos para aplicar estilos css

Ejemplo:

#encabezado{

color: red

}

Este selector aplicará estidos a todas las etiquetas con la clase #encabezado.

Selector Universal:

Este selector aplica estilos a todo el documento html y se llama mediante "*"

Ejemplo:

*{

color: red

}

Este selector aplicará estidos a todas las etiquetas html;
