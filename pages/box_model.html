Que es el box model?

El box model en css es un concepto muy importante y que se usa bastante en css ya que define y representa como se verá cada elemento en cualquier página web

Cada elemento en html se considera una cajita , por lo que se aplicarán conceptos de alto, ancho, relleno(padding) y margen(margin)

Capas del Box Model

El modelo de caja consta de cuatro partes principales:

Contenido (Content): Es el área donde se muestra el contenido real del elemento, como texto, imágenes, etc.

Relleno (Padding): Es un espacio transparente alrededor del contenido, dentro de la caja. Puede ser especificado en píxeles, porcentajes u otras unidades.

Borde (Border): Es una línea que rodea el relleno y el contenido. Al igual que el relleno, el borde puede tener un ancho y un estilo definidos.

Margen (Margin):Es el espacio transparente fuera de la caja. Es el espacio que separa esta caja de otras cajas adyacentes.

Por lo tanto la formula para calcular el tamaño total de una etiqueta es:

Tamaño total= Contenido + Relleno + Borde + Margen

Que es el margin negativo?

El uso de márgenes negativos en CSS implica que se está moviendo un elemento fuera de su contenedor o superponiendo elementos vecinos. Se puede aplicar márgenes negativos para posicionar elementos de manera específica

Que es el desbordamiento vertical por margenes?

El colapso de márgenes verticales en CSS es un comportamiento específico cuando dos márgenes verticales adyacentes se encuentran. En lugar de sumarse, los márgenes colapsan, tomando el valor del margen más grande entre ellos o, si uno de ellos es negativo, el valor más pequeño.

Que es el centrado perfecto horizontal?

El centrado perfecto horizontal se refiere a la técnica de centrar un elemento horizontalmente dentro de su contenedor padre. Se puede lograr esto de varias maneras, pero una de las más comunes y simples es utilizando la propiedad margin: auto; en combinación con un ancho definido para el elemento que estás centrando.

Que es el selector universal "*" y en que casos se usa?

El selector universal * en CSS selecciona todos los elementos en una página HTML. Se utiliza para aplicar estilos globales que afectan a todos los elementos dentro del alcance del selector.

Uso: RESETAR ESTILOS:

Se puede utilizar * para aplicar un "reset" básico de estilos, estableciendo propiedades como margin y padding en cero, para asegurarte de que todos los elementos tengan márgenes y rellenos consistentes.

USO: ESTABLECER ESTILOS GLOBALES:

Si se desea aplicar un estilo global a todos los elementos, como un tipo de fuente predeterminado o un color de texto base, se puede usar el selector universal.
